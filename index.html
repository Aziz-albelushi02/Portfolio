About Me

Your friendly neighborhood programmer.

Hello, I’m Aziz, a cybersecurity-focused professional based in Kuwait and the UK. While my background started in software development, my interests and career direction have shifted strongly toward security, penetration testing, and risk assessment.

I work primarily with Python and JavaScript, using them for scripting, automation, and security-related tasks. I also have prior experience with Java, which helped build a solid foundation in object-oriented programming. Over time, I became particularly drawn to understanding how systems fail, how vulnerabilities are introduced, and how they can be realistically exploited and mitigated.

My main area of focus is web application penetration testing, where I am most comfortable working with vulnerabilities outlined in the OWASP Top 10. I regularly practice and sharpen these skills through CTF challenges and hands-on labs on platforms such as TryHackMe, Hack The Box, as well as in-real-life challenges, which have significantly strengthened my analytical thinking and attacker mindset.

Alongside security, I retain solid experience in web development, having designed and developed web pages using HTML, CSS, PHP, and Bootstrap. This development background allows me to approach security not only from an attacker’s perspective, but also with an understanding of how applications are built and where common design mistakes occur.

I also have experience in mobile application development using Flutter (Dart), adding to my overall technical skill set. I enjoy combining development knowledge with hands-on offensive security work and continuous self-learning, allowing me to approach systems with a deeper and more practical security perspective.

I have hands-on experience with machine learning, using Python and its libraries to build a machine-learning–based intrusion detection model aimed at detecting wireless injection attacks. This work gave me practical insight into how machine learning can support security monitoring and threat detection in real environments.

Possession of a strong interest in cryptography particularly in its practical and theoretical applications as I actively explore cryptographic challenges in CTFs, along with independent research in the field. I have also helped in the development of a cryptography-based encryption and decryption application using the Java SDK, and conducted research into quantum cryptography, which has further strengthened my understanding of modern and emerging cryptographic concepts.

Projects

Some of my recent works.

Java 2D game "Ghost Hunting"

A Pacman-based game is developed using Java and allows players to control their character using either the arrow keys or the WASD keys. Each game features unique, randomly generated maps, ensuring a fresh experience every time. As players progress through levels, the game's difficulty increases, presenting more challenging enemies and complex maze designs. Evidence Database Platform

The impact management-based collaboration (team) project website developed using PHP for the backend, with HTML, JavaScript, and Bootstrap for the frontend. It supports two user types, each with different sets of CRUD (Create, Read, Update, Delete) abilities, and features distinct pages tailored to each user type. A settings page is included to allow users to change their details or modify the web theme. Enhanced security measures are implemented for user creation and logging functionalities. File Processing application

The file processing application is a Java-based tool designed to streamline the categorization of textual data from structured files. It utilizes both IO and NIO file handling methods, providing a comprehensive understanding of Java APIs for file operations. The application implements logic to automatically sort data based on predefined criteria, outputting the segregated information into multiple files for improved data manageability. Additionally, it features enhanced error handling capabilities to ensure robust file processing under various file system conditions.

IOS mockup calculator

The IOS Calculator App is developed using Flutter, incorporating a Java plugin to ensure compatibility with Android phones. Real-Time Face Detection Application

The Real-Time Face Detection Application is a Python-based program developed using OpenCV. It detects and highlights faces in real-time video feeds, offering dynamic camera selection and visual feedback capabilities. This application enables users to choose different camera sources while providing immediate visual cues to indicate detected faces within the video stream.

Cryptography app

0xCrypt is a collaborative project with 0xA designed to help people intuitively learn cryptography. The app features a user-friendly interface and supports expanding with new ciphers, product ciphers, and custom cipher creation and saving. Built on an MVC framework, it separates encryption/decryption from the frontend, with cipher fragments calling relevant functions for a clean code structure. Firebase Authentication manages user accounts, while the Firebase Realtime Database stores and restores user snapshots, ensuring users can seamlessly continue their work from their last session.

Machine learning for the detection of wireless injection attacks (Prototype)

The machine learning application for detecting wireless injection attacks involves creating models to identify Man-in-the-Middle and deauthentication attacks. Utilizing a CSV-based dataset containing malicious features for both types of attacks, the data was meticulously cleaned and transformed to ensure high accuracy of the models. Each step of the process has been thoroughly documented to guarantee reproducibility, making the approach both robust and transparent.

Diabetes Prediction using Machine Learning

This report uses the Weka tool and Python to develop AI solutions for diabetes classification using a dataset from Kaggle.com. Attributes from the dataset, originally shared by Alex Teboul, help train AI models to predict diabetes onset, suggesting lifestyle changes. Libraries such as SVM, Pandas, and Decision Tree classifiers were employed, along with sklearn for metrics and seaborn/matplotlib for visualization. The dataset was split 80/20 for training/testing, achieving a 76% accuracy. A numpy calculation function aided in model evaluation and prediction.
